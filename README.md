# MAGIC GAME

Projeto Teste de um Jogo de Magos em Java, proposto pelo Isaac Souza
--
O jogo é jogavel no Terminal do VsCode, sendo varias funcionalidades que podem ser feitas no jogo, mecanicas de combate em equipe e solo, alem de uma integração de diferentes tipos de magos.
